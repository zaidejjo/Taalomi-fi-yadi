## Overview

**Ta3lemi Fi Yadi** (which translates to "Learning in My Hands") is a modern, user-friendly online educational platform designed to make learning accessible and engaging for everyone. The platform offers an intuitive interface, structured courses, and interactive features to enhance the learning experience.

This project demonstrates a full-stack web application, combining a secure backend, responsive frontend, and easy deployment.

---

## Features

- **Interactive Courses:** Users can browse, enroll, and follow structured courses.
- **User Authentication:** Secure login, registration, and account management.
- **Role-based Access:** Different access levels for students, instructors, and admins.
- **Responsive Design:** Fully mobile-friendly and works on all devices.
- **RTL Support:** The platform supports Arabic and other right-to-left languages.
- **Secure and Scalable:** Built with modern best practices for security and performance.
- **Live Demo:** Access the platform online [here](https://ta3lemi-fi-yadi.onrender.com/).
